2. Application Contents and Usage

Christ AI is an AI assistant that can answer religious questions and share religious information. The application consists of a single main chat interface

  • Ask Chris: You can ask the AI assistant questions about faith, christianity, the bible and more through this chat interface via typing or speaking to the microphone.

Inside the application, free users are provided with 10 message credits that can be used daily. Premium users are provided with 100 message credits that can be used daily (90 classic model + 10 advanced reasoning model).
