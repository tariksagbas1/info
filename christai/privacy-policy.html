Privacy Policy

1. Introduction

As the Christ AI application, protecting our users' privacy is one of our most important priorities. This privacy policy explains how your personal data is collected, used, and protected when using our application. Christ AI does not require any account information or identity data from users. However, during application use, an anonymous user identifier, message usage statistics, and email address shared for support requests may be processed due to technical requirements. This data is used only for limited purposes such as service operation, security, and cost tracking, and is not shared with third parties.

2. Collected Data

Our application collects the following data:

  • Anonymous User Identifier: A unique identifier for in-app functionality (does not contain personal information)
  • Daily Message Quotas: Remaining message counts for classic and reasoning models
  • Total Message Statistics: Total message counts sent for both categories
  • Token Usage Information: Token consumption data for tracking AI API costs
  • Account Level: Free or premium membership status
  • (Optional) Email Address: Personal email address is requested for support requests.

Important: Chat history, voice recordings, and personal preferences are stored only on your device (locally) and are never sent to any server. Personal information such as name, surname or phone number are never collected.

3. Data Usage Purposes

Collected data is used for the following purposes:

  • Tracking and enforcement of daily message limits
  • Calculation and control of AI API costs
  • Providing application functionality
  • Creating usage statistics
  • Resolving technical issues
  • Email address, when shared, is used only for technical support communication purposes.

4. Data Storage

Your data is stored for the following periods:

  • Anonymous User Identifier: As long as the application is used
  • Daily Message Quotas: Stored daily.
  • Token Usage Information: Stored indefinitely for cost tracking.
  • Local Data: Chat history and preferences are stored only on your device. Such data is never collected, stored on servers, or recoverable.

5. Data Sharing

Your anonymous data is not shared with third parties. Data sharing may only occur in the following cases:

  • Legal obligations
  • With the user's explicit consent
  • OpenAI - Message content for AI responses (your questions are sent only to receive answers, not stored)

Note: No data is shared with Google AdMob. Non-personalized general ads are shown.

6. Security

We take the following measures to ensure the security of your data:

  • Secure database management
  • Regular security updates
  • Access controls and authorization
